# Demo: Zone Setup

Demonstrates ZDHG with a pre-configured scene.

## Setup
1. Import this sample via Package Manager
2. Open the DemoZone scene (Scenes/DemoZone.unity)
3. The scene has:
   - A baked NavMesh
   - GameObjects tagged EnemySpawn / ResourceSpawn
   - Two ZoneDefinition assets (Town / Wilderness)
4. Open Window > Live Game Dev > Zone Density Heatmap
5. Click Add All Zone Assets in the Zones panel
6. Click Generate
7. Toggle the Scene View overlay to see the heatmap live
